=== VS Code 远程连接后显示 GUI 指南 ===

---------- 本地 Windows 操作 ----------

Step 1: 下载安装 VcXsrv
  https://sourceforge.net/projects/vcxsrv/

Step 2: 双击 XLaunch 启动
  一路下一步，最后勾选 "Disable access control"，完成。
  右下角出现 X 图标即成功。

Step 3: 配置 VS Code SSH
  Ctrl+Shift+P → Remote-SSH: Open SSH Configuration File
  在 Host 配置里加这两行：
    ForwardX11 yes
    ForwardX11Trusted yes
  保存后重新连接远程服务器。

---------- 远程服务器操作 ----------

Step 4: 验证 DISPLAY
  echo $DISPLAY
  看到 localhost:10.0 之类即表示通。

Step 5: 启动 GUI
  cd ~/pipeline_tool/AscendDevTool
  source ascenddevtool/bin/activate
  python gui/main.py

  GUI 窗口会弹到你本地 Windows 桌面上。
